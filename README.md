# 🚀 Konecta - N-Tier ASP.NET Core Web API

## 📌 Overview
**Konecta** is a **structured N-Tier ASP.NET Core Web API** following best practices for **separation of concerns**.  
It is designed to be **scalable, maintainable, and extensible**.  

This API exposes a **GET endpoint (`/api/products`)** that returns mock product data.  
It uses **Swagger** for testing and **Dependency Injection (DI)** for clean architecture.

---

## 📌 Features
✅ **N-Tier Architecture** (Separation of Concerns)  
✅ **ASP.NET Core Web API**  
✅ **Service Layer for Business Logic**  
✅ **Data Layer for Repository Implementation**  
✅ **Swagger UI for API Testing**  
✅ **Dependency Injection (DI)**
